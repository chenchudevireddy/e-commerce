========================================
SHOPHUB E-COMMERCE APPLICATION
Angular-based Flipkart/Amazon Style App
========================================

📋 QUICK START GUIDE

1. INSTALL DEPENDENCIES:
   npm install

2. START THE APPLICATION:
   npm start
   
   Or use the startup script:
   - Windows: Double-click "startup.bat"
   - Mac/Linux: Run "bash startup.sh"

3. OPEN IN BROWSER:
   http://localhost:4200

4. EXPLORE THE APP:
   ✓ Browse products on the home page
   ✓ Filter by categories (Electronics, Mobiles, etc.)
   ✓ Search for products
   ✓ View product details
   ✓ Add items to cart
   ✓ Checkout with multi-step form
   ✓ Track your order

🔧 TROUBLESHOOTING

Problem: Port 4200 already in use
Solution: npm start -- --port 4300

Problem: npm install fails
Solution: 
  - Delete node_modules folder
  - Delete package-lock.json
  - Run: npm install again

Problem: Build errors
Solution: npm run build

Problem: Module not found errors
Solution: npm install

📖 HOW TO USE THE APP

1. BROWSE PRODUCTS:
   - Go to home page
   - Click on a category or "Start Shopping"
   - See all products in grid view

2. FILTER & SEARCH:
   - Use sidebar to filter by category
   - Use search bar to find specific products
   - Sort by price, rating, or discount

3. VIEW PRODUCT DETAILS:
   - Click on any product card
   - See full description, ratings, reviews
   - Check delivery information
   - Select quantity

4. ADD TO CART:
   - Click "Add to Cart" button
   - View cart anytime from header icon

5. CHECKOUT:
   - Go to cart
   - Click "Proceed to Checkout"
   - Step 1: Enter shipping details
   - Step 2: Enter delivery address
   - Step 3: Enter payment information
   - Complete order

========================================

💾 DATA STORAGE

The shopping cart is saved in browser localStorage:
- Cart persists even after closing the browser
- Clear cache to reset cart
- Each product is stored with quantity

========================================

🎯 CUSTOMIZATION

To add more products:
1. Open: src/app/services/product.service.ts
2. Find the "products" array
3. Add new product objects following the existing format

To change colors:
1. Modify src/styles.scss
2. Or edit individual component SCSS files

To add new components:
ng generate component components/component-name
