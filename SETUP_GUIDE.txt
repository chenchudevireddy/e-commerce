                            🎯 HOW TO USE THE APPLICATION
════════════════════════════════════════════════════════════════════════════════

SCENARIO 1: Browse All Products
─────────────────────────────────
1. Open home page
2. Click "Start Shopping" button
3. See all 16 products in grid view
4. Use sidebar to filter by category
5. Use dropdown to sort by:
   - Price (low to high or high to low)
   - Rating (highest first)
   - Discount (most discount first)

SCENARIO 2: Search for a Product
───────────────────────────────────
1. Use the search bar at the top
2. Type product name (e.g., "Gaming Laptop")
3. Press Enter or click search icon
4. See filtered results
5. Click on any product to view details

SCENARIO 3: Buy a Product
─────────────────────────
1. Find a product (browse or search)
2. Click on the product card
3. View full details and related products
4. Select quantity
5. Click "Add to Cart"
6. Cart count updates in header
7. Click cart icon to view cart
8. Click "Proceed to Checkout"
9. Fill in details for 3 steps
10. View order confirmation

SCENARIO 4: Manage Shopping Cart
──────────────────────────────────
1. Click cart icon in header
2. See all items with prices
3. Update quantities using +/- buttons
4. Remove items with trash icon
5. View order summary:
    - Subtotal
    - Discount (automatic)
    - Delivery charges (FREE above ₹500)
    - Final total
6. Continue shopping (back to products)
7. Or Proceed to Checkout

ADD NEW PRODUCTS:
────────────────
1. Open: src/app/services/product.service.ts
2. Find the "products" array
3. Add new object following this format:

  id: 17,  // Next ID
  name: 'Product Name',
  price: 9999,

  reviews: 100,

                            📝 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

PROBLEM: npm install fails
SOLUTION:
  1. Delete node_modules folder
  2. Delete package-lock.json
  3. Clear npm cache: npm cache clean --force
  4. Run: npm install

PROBLEM: Port 4200 already in use
SOLUTION: npm start -- --port 4300

PROBLEM: Components not appearing
SOLUTION:
  1. Check if component is declared in app.module.ts
  2. Check if component selector is correct
  3. Clear browser cache (Ctrl+Shift+Delete)

PROBLEM: Cart data lost after refresh
SOLUTION:
  - Check if localStorage is enabled in browser
  - Check browser console for errors (F12)

PROBLEM: Styles not applying
SOLUTION:
  1. Check SCSS syntax
  2. Rebuild: npm start (it auto-rebuilds)
  3. Hard refresh browser (Ctrl+Shift+R)
